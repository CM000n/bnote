> Update BNote
Update is possible from version 3.2.2 onwards.
Please make sure to copy ALL files onto your web server.

! IMPORTANT !
Please add the following configuration to your BNote main config.
File:  BNote/config/config.xml
<Software>
	[...]
	<Theme>default</Theme>
	<Logo>BNote_Logo_white_transparent_44px.png</Logo>
	[...]
</Software>
