Das aktuelle Release dieser Software ist Version 2.5.
Auf GitHub sind genaue Angaben zu Features und Bugs die mit dieser Version behoben sind vorhanden. Hier jedoch eine kleine Übersicht mit den wichtigsten Änderungen:

Probenverwaltung

Aufgaben

Bandleiter können nun Aufgaben für alle Kontakte einer Gruppe erstellen.

Share

Ist auf dem Server die PHP zip-Erweiterung installiert und lässt die Konfiguration zip-Downloads eines Ordners zu, dann kann ein Ordner als Zip-Archiv heruntergeladen werden.

Fehlerbehebung

Außer den zuvor genannten Änderungen wurden von Testern und Benutzern gemeldete Fehler behoben.

Hilfe

Die Hilfeseiten wurden beträchtlich erweitert und es wurden Sicherheitshinweise hinzugefügt.

Genauere Hinweise zu den Änderungen und Neuerungen der einzelnen Versionen finden sich beim jeweiligen Update in der Datei release_notes.txt.
